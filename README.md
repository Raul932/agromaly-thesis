# AgriTech AI - Proactive Agricultural Management

Bachelor Thesis Project - Artificial Intelligence @ Babeș-Bolyai University (UBB FMI)

##  Project Overview
An intelligent, cross-platform ecosystem designed to help farmers monitor crop health, process geospatial data, and receive proactive agronomic advice via Agentic AI (RAG).

##  Architecture
* **Backend:** Python, FastAPI, SQLAlchemy
* **Database:** PostgreSQL + PostGIS (Geospatial)
* **AI/ML:** PyTorch (LSTM Autoencoder), LangChain, ChromaDB
* **Infrastructure:** Docker, Celery, Redis
